## 💻 Sobre o Projeto

Este projeto serve como a minha montra profissional. A intenção ao desenvolvê-lo com tecnologias fundamentais (Vanilla Web) foi demonstrar um domínio sólido de manipulação do DOM, estruturação semântica de interfaces e estilização avançada com CSS moderno, sem a dependência inicial de *frameworks*.

## ✨ Funcionalidades

- **Alternância de Tema (Dark/Light Mode):** Implementação nativa através de manipulação de atributos e variáveis CSS.
- **Design Totalmente Responsivo:** Layout fluido que se adapta desde dispositivos móveis a ecrãs *ultrawide*, incluindo um menu hambúrguer interativo.
- **Navegação Suave (Smooth Scrolling):** Transições de página elegantes entre as secções do portfólio.
- **Catálogo de Projetos:** Secção dedicada à apresentação detalhada de aplicações desenvolvidas.

## 🛠️ Tecnologias Utilizadas

- **[HTML5]** - Estrutura semântica e acessível
- **[CSS3]** - Estilização avançada (Flexbox, variáveis e media queries)
- **[JavaScript (ES6+)]** - Lógica de interatividade e manipulação do DOM
- **[Font Awesome]** - Ícones vetoriais da interface

## 🚀 Como visualizar

Como o projeto é composto por ficheiros estáticos (Vanilla), a execução é extremamente simples:

1. Clone este repositório:
`git clone https://github.com/jeancarloslima/portfolio-jean-carlos-lima.git`
2. Aceda à pasta do projeto:
`cd portfolio-jean-carlos-lima`
3. Abra o ficheiro `index.html` diretamente no seu navegador, ou utilize a extensão *Live Server* do VS Code para uma melhor experiência
